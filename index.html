Les boutons Explication
Pour commencer, cliquez sur ce bouton pour charger votre fichier. Attention, seuls les fichiers au format .txt sont supportés.

Avec ce bouton, vous obtiendrez un tableau avec les mots les plus fréquents du texte triés dans l'ordre décroissant.
Ce bouton permet de rechercher un mot dans un texte.
Cette fonction vous permet de rechercher les contextes gauches et droits d'un mot spécifique. Pour cela, vous devez préciser votre mot dans le pôle, puis entrez la longueur pour décider de la taille des contextes souhaités.
En bas de l'analyseur, vous trouverez une animation et pour activer la réponse il faut cliquer sur le bouton. Il permet de compter les types de phrase dans le texte.
